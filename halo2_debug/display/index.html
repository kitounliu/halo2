Structs§

  • ExprDisp
    Wrapper type over Expression that implements Display with nice output. The formatting of the Expression::Variable case is parametrized with the second field, which take as auxiliary value the third field. Use the constructor expr_disp to format variables using their Display implementation. Use the constructor expr_disp_names for an Expression with V: VarMid to format column queries according to their string names.
